CodeSystem: DIPS care relation code system

Official URL: urn:AuditEventHL7Norway/CodeSystem/carerelation Version: 0.1.0
Active as of 2023-02-06 Computable Name: DIPSCareRelationCS

Dette er maler med ulike begrunnelser som kan benyttes som grunnlag for tilgang til pasientens journal ved sykehus som benytter DIPS som journalsystem.

This Code system is referenced in the content logical definition of the following value sets:

  • PurposeOfUseVS
