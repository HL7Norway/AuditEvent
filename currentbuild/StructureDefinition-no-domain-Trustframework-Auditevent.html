Resource Profile: no-domain-Trustframework-Auditevent

Official URL: http://hl7.no/fhir/StructureDefinition/no-domain-Trustframework-Auditevent Version: 0.9.6
Active as of 2024-09-10 Computable Name: NoDomainTrustframeworkAuditEvent

This is the main profile that describes the mapping of Norwegian Trust Framework attributes ("Tillitsrammeverk") to AuditEvent Resource.

Two extentions is added to AuditEvent R4 resource to be forward compatible with R5:

  • _encounter
  • _patient

Usage:

  • Examples for this Resource Profile: AuditEvent/NOBasisAuditeventInstance1, AuditEvent/NOBasisAuditeventInstance2 and AuditEvent/NOBasisAuditeventInstance3
